### **🛠️ Installation & Setup**
1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/ZainAliAwan/CodeAlpha_Chatbot.git](https://github.com/ZainAliAwan/CodeAlpha_Chatbot.git)
    ```
2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run Application:**
    ```bash
    streamlit run app.py
    ```
